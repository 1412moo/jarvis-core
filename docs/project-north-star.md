# Project North Star

## 1. 프로젝트 목적
- 이 프로젝트는 개인용 Jarvis형 AI 오케스트레이션 시스템의 기준(지휘/기록)을 정의한다.
- 목표는 입력 → 기록 → 승인 → 실행으로 이어지는 개인 자동화 흐름을 안정적으로 구축하는 것이다.

## 2. 현재 단계
- 현재는 Discord 기반 입력/상태 관리에 더해, 제한된 실행 레이어를 함께 검증하는 단계다.
- Discord를 입력 인터페이스로 사용하며, markdown 기반 task 저장 구조를 운영한다.
- 기본 흐름은 `입력 -> 기록(memory/tasks) -> 승인(/approve) -> 제한 실행(/run, /retry)`으로 유지한다.
- 실행은 범용 명령 실행이 아니라 화이트리스트로 제한된 안전 실험 범위에서만 동작한다.

## 3. 장기 목표
- 홈서버 기반 상시 동작 개인 자동화 시스템으로 확장한다.
- 승인(`/approve`) 이후 실행 레이어를 점진 확장해 실제 작업 수행까지 이어지게 만든다.
- 채널은 Discord 중심에서 필요 시 웹/전용 컨트롤러로 확장하되, 동일한 오케스트레이션 기준을 유지한다.
