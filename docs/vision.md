# Vision

## 장기 목표
`jarvis-core`는 다중 프로젝트 운영을 위한 중앙 지휘 저장소로서,
아래 역할을 안정적으로 수행하는 것을 목표로 한다.

- 운영 규칙의 단일 기준점(Single Source of Truth) 제공
- Codex/에이전트 작업 지침 표준화
- 보고/기록 체계 표준화
- 서브 저장소와의 역할 분리를 통한 유지보수성 확보
- 추후 Discord/웹 인터페이스와 연동 가능한 오케스트레이션 중심축 제공

## 단계별 접근
- 1단계: 구조/문서 중심 부트스트랩
- 2단계: 최소 자동화(검증/정리 스크립트)
- 3단계: 오케스트레이터/어댑터 확장
- 4단계: 운영 연동(외부 인터페이스, 워크플로우)

## 성공 기준(초기 단계)
- 저장소 역할이 문서로 명확히 정의되어 있다.
- 작업 규칙과 보고 규칙이 일관되게 적용된다.
- 추후 기능 확장을 방해하지 않는 느슨한 구조를 유지한다.
